Worker
1. Network
	1. Обнаружение себеподобных (UDP) и ответ на попытки обнаржуения
		1. нужно определиться с используемымит портами
		2. нужно сообщать метрику self test всем
	2. Поднятие соединение ко всем обнаруженным
		1. Исключение формирование двойных соединений (например через сравнение рандомных токенов при обнаружении по UDP)
		2. Проверка скорости соединения и сообщение её второй стороне 
	3. Принятие/отправка данных другим узлам
		1. вероятно тип отправки (текстом/бинарём) надо определять динамически
	4. Принятие/отправка данных от внешнего клиента
2. Computing core
	1. Принятие данных и функции обработки
	2. применение функции обработки к каждому элементу данных
3. Self Test
	1. Выполнение 1000(?) каких-то операций (например tg(x++)) с измерением времени выполнения. 
4. Control Center
	1. получение данных от внешнего клиента
	2. распределение задачи по worker-ам
		1. расчёт необходимой доли задачи на worker-а
		2. "разрезание" пакета данных и передачу частей соседям
		3. сбор выполненных частей с сохранением порядка записей
	3. выдача результата клиенту
Client
1. Network
	1. Обнаружение worker-ов
	2. Подключение к указанному worker-у
	3. Передача/приём данных
2. Первичный парсинг задачи (обр польская нотация?)
3. Взаимодействие с юзером
